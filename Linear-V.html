int :: Int -> TypeQSource

This can be used to generate a template haskell splice for a type level version of a given int.

This does not use GHC TypeLits, instead it generates a numeric type by hand similar to the ones used in the "Functional Pearl: Implicit Dimurations" paper by Oleg Kiselyov and Chung-Chieh Shan.
